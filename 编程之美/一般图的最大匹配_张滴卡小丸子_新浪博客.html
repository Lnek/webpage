一般图的最大匹配

(2012-02-22 21:08:48)
转载▼
标签：

杂谈

 分类： 学习笔记
在最一般的情况下，给定一个无向图G=(V,E)，最大匹配问题是要找出一个E的子集M，它具有最大数量的不交迭边，即在M中任何两条边没有一个共同顶点。这个问题来自许多实际应用，特别是在通信和调度领域。
对于下图求最大匹配。
编程运行，过程描述如下：
找到路径：ab，当前匹配为：{ (a,b) }
找到路径：dc，当前匹配为：{ (a,b), (c,d) }
找到路径：fe，当前匹配为：{ (a,b), (c,d), (e,f) }
找到路径：hg，当前匹配为：{ (a,b),(c,d), (e,f), (g,h) }
找到路径：ji，当前匹配为：{ (a,b), (c,d), (e,f), (g,h), (i,j) }
找到路径：lk，当前匹配为：{ (a,b), (c,d), (e,f), (g,h), (i,j), (k,l) }
找到路径：pijklu，当前匹配为：{ (a,b), (c,d), (e,f), (g,h), (i,p), (j,k), (l,u) }
找到路径：rq，当前匹配为：{ (a,b), (c,d), (e,f), (g,h), (i,p), (j,k), (l,u), (q,r) }
找到路径：ts，当前匹配为：{ (a,b), (c,d), (e,f), (g,h), (i,p), (j,k), (l,u), (q,r), (s,t) }
找到路径：vulkjipo，当前匹配为：{ (a,b), (c,d), (e,f), (g,h), (i,j), (k,l), (o,p), (q,r), (s,t), (u,v) }
上图最大匹配为：
{ (a,b), (c,d), (e,f), (g,h), (i,j), (k,l), (o,p), (q,r), (s,t), (u,v)
