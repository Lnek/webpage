Vim for Eclipse

(2011-09-09 13:04:48)
转载▼
标签：

java

spring

vim

 分类： Java
最近在做关于安全的测试，使用到了Java的spring框架，又重新开始写点Java代码再熟悉下。由于最近经常使用Vim，感觉十分的方便就想找个eclipse的Vim插件来用用，减轻鼠点标依赖症。发现了一个很好的Vim插件---vimPlugin。这是一个收费软件，但是很容易就进行了破解，只需要在eclipse的根目录下面创建一个viPlugin.license的文件就搞定了，这个文件里面的内容为Xq@a*4。如果不能确定将这个文件放在哪里的话，可以直接安装好插件启动eclipse，eclipse自动会提醒你找不到这个license文件的路径，只要按着这个路径去创建这个license文件就搞定了。
此外，顺便说下eclipse安装插件的方法，很早以前的用的时候研究了很久才搞明白的，这次又搞了一小会。其实方法还是比较简单的，在eclipse的根目录下新建一个名为links的文件夹，进入文件夹新建一个文件viPlugin.txt。这个文件的后缀可以是任意的（除了exe，bat这种的）比如txt，link，links什么的都是可以的，只要在于文件名，文件名必须和你的插件的最外层文件夹名字一致。看下图

viPlugin是这整个插件最外一层目录，所以我们在links底下的文件名叫做viPlugin.txt。然后在文件中输入path=“路径”，就可以了。值得注意的是这里的路径需要使用转义（个人觉得是转义的意思），即应该使用双斜杠比如C:\\users。
