centos 6.x samba 加入windows 域

分类： linux 2012-09-04 21:20 459人阅读 评论(0) 收藏 举报
centoswindowsdomainpasswordsserversecurity

软件需求：

krb5-workstation-1.9-33.el6_3.2.x86_64
krb5-libs-1.9-33.el6_3.2.x86_64
krb5-devel-1.9-33.el6_3.2.x86_64
pam_krb5-2.3.11-9.el6.x86_64


samba-3.5.10-125.el6.x86_64
samba-client-3.5.10-125.el6.x86_64
samba-winbind-clients-3.5.10-125.el6.x86_64
samba-winbind-3.5.10-125.el6.x86_64
samba-common-3.5.10-125.el6.x86_64
samba4-libs-4.0.0-23.alpha11.el6.x86_64

编辑/etc/krb5.conf
[libdefaults]
 default_realm = EXAMPLE.COM
 dns_lookup_realm = false
 dns_lookup_kdc = false
 ticket_lifetime = 24h
 renew_lifetime = 7d
 forwardable = true


[realms]
 EXAMPLE.COM = {
  kdc = IP:88 (domain controller ip)
  admin_server = IP:749
  default_domain = EXAMPLE.COM
 }


[domain_realm]
 .example.com = EXAMPLE.COM
 example.com = EXAMPLE.COM


编辑/etc/nsswitch.conf
passwd:     files winbind
shadow:     files winbind
group:      files winbind

编辑/etc/resolv.conf

nameserver  IP

===========================================

编辑/etc/hosts

127.0.0.1 hostname.example.com hostname

编辑/etc/sysconfig/network

HOSTNAME=hostname

============================================net_update_dns_internal: Failed to connect to our DC!DNS update failed!

编辑/etc/samba/smb.conf

#======================= Global Settings =====================================
# ----------------------- Network Related Options -------------------------
        workgroup = EXAMPLE
        server string = Samba Server Version %v
        netbios name = hostname
# ----------------------- Domain Members Options ------------------------
        security = ads
        passdb backend = tdbsam
        realm = example.com
        password server = IP

        encrypt passwords = yes
        idmap uid = 16777216-33554431
        idmap gid = 18777216-33554431
        template shell = /bin/bash
        template homedir = /home/%U
        winbind use default domain = true
        winbind offline logon = true
        winbind enum groups = yes
        winbind enum users = yes
        winbind separator = /


加入域 net ads join -U administrator

离开域 net ads leave -U administrator

加域前先ntpdate IP一下，与域控同步时间防止出错

用域用户登录时如果出错，用setup命令检查下图选项是否已选择
