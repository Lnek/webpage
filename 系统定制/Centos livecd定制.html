Centos livecd定制
2009-09-18 11:10
1.安装一Centos系统
2.下载配置文件https://projects.centos.org/trac/livecd/browser/trunk/centos-livec-destop.ks
然后根据自己需求定制KS文件：
lang en_US.UTF-8
keyboard us
timezone US/Eastern
auth --useshadow --enablemd5
selinux --enforcing

repo --name=a-base    --baseurl=ftp://192.168.11.99/
repo --name=a-updates --baseurl=ftp://192.168.11.99/
repo --name=a-extras  --baseurl=http://mirror.centos.org/centos/5/extras/$basearch
repo --name=a-live    --baseurl=http://www.nanotechnologies.qc.ca/propos/linux/centos-live/$basearch/live
xconfig --startxonboot
part / --size 4096
services --enabled=cups,haldaemon,mcstrans,NetworkManager,portmap,restorecond --disabled=anacron,auditd,bluetooth,cpuspeed,gpm
,hidd,ip6tables,mdmonitor,microcode_ctl,netfs,network,nfslock,readahead_early,readahead_later,rpcgssd,rpcidmapd,sshd


%packages
syslinux
kernel

@base-x
#packages removed from @base-x
-bitstream-vera-fonts
-linuxwacom
-rhgb
-vnc-server
-xorg-x11-server-Xnest
-xorg-x11-twm
-xterm
#@base-x <end of package list>

@core
#packages removed from @core
-ed
-gnu-efi
-libhugetlbfs
#@core <end of package list>

# Other packages we don't want to include in the Live CD
-*debuginfo
-bind-libs
-compat*
-exim
-gamin-python
-nscd
-oddjob-libs
-procmail
-python-ldap
-rmt
-tclx
-yp-tools


# other usefull packages
Cluster_Administration-en-US
Deployment_Guide-en-US
Global_File_System-en-US
Virtualization-en-US
busybox
mailx
memtest86+
patch
yum-fastestmirror
yum-metadata-parser
yum
firefox
httpd
vsftpd
openssh-server
openssh
openssh-askpass
openssh-clients


# LiveCD bits to set up the livecd and be able to install
# Installation from the livecd requires anaconda >= 11.2.0.66
#anaconda
#anaconda-runtime
livecd-installer


%post
.......
后面的内容保持不变。
3.在centos系统上配置一FTP服务器作为本地源，这样下载速度会快很多。
4.运行LANG=C livecd-creator --config=centos-livecd-desktop.ks --fslabel=CentOS-5.3-i386-LiveCD命令制作livecd
5.可以在ks文件的post部分加入脚本，使得livecd从text mode启动，并修改/etc/X11/xinitrd/Xclients文件，在其中加入exec "/usr/bin/firefox"等，让其启动firefox作为主界面；同时修改$HOME/.mozilla/firefo/2f6fchy8.default/chrome目录下userChrome.css文件，改变firefox外观（隐藏主菜单等）：
#toolbar-menubar{
display: none !important;
}

#navigator-toolbox{
display: none !important;
}

#menubar, toolbox, toolbar {border-style: none !important; }

#tabbar {border-style: none !important; }
