定制centos livecd

(2009-03-11 15:24:54)
转载▼
标签：

杂谈

 分类： PXE/Kickstart
网络上已经有了各种版本的livecd,只是用centos用久了，比较习惯它。centos官方有livecd，不过我想拿来pxe启动维护用，不过那个太大了，不方便。于是google一通，发现centos的wiki里已经有很好的方法自己定制livecd了。下边简单记述一下，基本是按我的理解翻译的官方说明,
 
装好一台CentOS 5的机器，别用CentOS 4.x，也别用LVM，并且禁用SElinux (因为制作livecd的脚本对python和yum的版本有要求；还有用了LVM制作时会报错，如果不信你可以试试，呵呵)
 
1.创建 /etc/yum.repos.d/centos-livecd.repo
# Name: CentOS LiveCD repository
[livecd]
name = CentOS $releasever - LiveCD
baseurl = http://www.nanotechnologies.qc.ca/propos/linux/centos-live/$basearch/live
enabled=1
protect=0
gpgkey = http://www.nanotechnologies.qc.ca/propos/linux/RPM-GPG-KEY-PGuay

2.安装livdcd-tools



# yum --nogpgcheck install livecd-tools syslinux



 



3.定制你自己的kickstart脚本，这里用的kickstart脚本和自动安装时使用的是一样的，语法参考kickstart手册



 



4.制作livecd 光盘镜像



# LANG=C 



# livecd-creator --config=centos-livecd-minimal.ks --fslabel=CentOS-minimal



 



这里--config后边用你自己的ks文件，fslable随你自己喜欢，生成好的livdcd image文件名就是fslable那里定义的，例如CentOS-minimal.iso



 



这样就完事了！
