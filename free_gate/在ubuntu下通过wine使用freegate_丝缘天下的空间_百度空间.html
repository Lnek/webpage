在ubuntu下通过wine使用freegate

发现越来越多到网站被墙了，特别是于谷歌相关到，比如android开发网站都被墙而无法打开，这是我无法容忍到，最近都在使用ubuntu，因此不知道 自由门如何使用。如果是在windows环境下，我还可以通过自由门来打开twitter，facebook等被墙到网站。那么在ubuntu下如何打开 一些只能在windows下才能被打开到文件呢。这就是我们今天要介绍到内容，通过wine来打开windows程序。

首先是wine到安装，具体见：
http://forum.ubuntu.org.cn/viewtopic.php?t=107816 。
我将我到操作过程列出如下。

1、首先运行
wget -q http://wine.budgetdedicated.com/apt/387EE263.gpg -O- | sudo apt-key add -
添加GPG钥匙

2、安装 WineHQ 发布的最新 deb 包
sudo apt-get update
sudo apt-get install wine

3.配置 winecfg
运行 wine 配置程序 ，不要用 sudo 或者 root 权限，直接使用命令：winecfg 。
这样会建立 wine 的运行文件夹 /home/user/.wine .里面有wine的注册表文件和虚拟的C盘；稍等片刻，就会出现一个面板，这就是 wine 的一些基本的配置。

4.DLL设置

在真实的 windows 系统中从 C:\WINDOWS\systenm32 里复制 mfc42.dll,msvcp60.dll, riched20.dll,riched32.dll 这几个文件到 /home/用户名/.wine/drive_c/windows/system32 文件里，需要覆盖时确定。其他dll文件不要随便覆盖，要做备份。

5.字体设置
1、从 Windows 目录下的 Fonts 里的 simsun.ttc 复制到 /home/user/.wine/drive_c/windows/fonts 里面。
把下面的代码保存为 zh.reg ，然后终端执行 regedit zh.reg 。 我这里将zh.reg放在/home/user1 目录下，这样直接在teminal终端中运行命令：regedit zh.reg。
zh.reg代码如下

REGEDIT4
[HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\FontSubstitutes]
"Arial"="simsun"
"Arial CE,238"="simsun"
"Arial CYR,204"="simsun"
"Arial Greek,161"="simsun"
"Arial TUR,162"="simsun"
"Courier New"="simsun"
"Courier New CE,238"="simsun"
"Courier New CYR,204"="simsun"
"Courier New Greek,161"="simsun"
"Courier New TUR,162"="simsun"
"FixedSys"="simsun"
"Helv"="simsun"
"Helvetica"="simsun"
"MS Sans Serif"="simsun"
"MS Shell Dlg"="simsun"
"MS Shell Dlg 2"="simsun"
"System"="simsun"
"Tahoma"="simsun"
"Times"="simsun"
"Times New Roman CE,238"="simsun"
"Times New Roman CYR,204"="simsun"
"Times New Roman Greek,161"="simsun"
"Times New Roman TUR,162"="simsun"
"Tms Rmn"="simsun"

wine配置完成。

6.运行自由门
右击自由门程序fg701p.exe，然后选择“使用 Wine Windows Program Loader 打开”。
这时候可能弹出一个ieexploer.exe的错误，这是因为在ubuntu下没有IE，因此我们在自由门“通道”选项卡中选择“经典模式（浏览器不需要设置代理）”。这样就不会报错误了。然后再次运行自由门，就能打开自由门主页了。
还有一点，我们在自由门“内容”选项卡中看到到内容有时是不同到，举例来说，http://127.0.0.1:8580有时候是http://127.0.0.1:8581，即每次打开自由门到端口号不是确定到，这影响到我们下面到AutoProxy设置。

7.安装AutoProxy
在firefox到“工具->添加附件”中搜索AutoProxy，然后安装。
安装完以后在“工具->AutoProxy首选项”中进行代理设置，在“代理服务器“中我们点击“选择代理服务器”，默认设定为“Free Gate”，然后点击“编辑代理服务器”，我们查看Free Gate这个代理的默认代理主机：127.0.0.1，端口为8580，在前面我们提到自由门到端口号不一定就是8580，因此我们如果想直接通过 firefox翻墙到话，就需要在这里进行具体设置，如果自由门到“内容”选项卡中端口号显示为8581，那么我们这里需要将端口改为8581。
在设定完上面这些内容以后，我们就能够翻墙上网了。
