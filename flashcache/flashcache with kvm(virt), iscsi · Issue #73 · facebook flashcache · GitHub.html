check-ict commented August 22, 2012

I have the same issue. I use flashcache with a 2TB eco drive. Without flashcache it gives 170 IOps. With flashcache on host it gives 6000 IOps in bonnie++. In the VM (with or without flashcache enabled) it will not get higher then normal/raw disk speed, which is around 170 IOps. I test this with OpenVZ and KVM and Bonnie++.

Please let me know if there is a way to increase the VM speed!

check-ict commented August 22, 2012

I just fixed it by putting cache=writeback in the KVM VM config. Now the performance is the same (near) the host performance!

check-ict commented August 23, 2012

Another importent note. Windows VM's need Virtio AND cache=writeback disks to perform well with flashcache.
With IDE (and writeback) = 400 IOps and 5MBps IOmeter
With Virtio (without writeback) = 700 IOps and 10 MBps IOmeter
With Virtio (and writeback) = 9500 IOps and 115 MBps IOmeter !!!
