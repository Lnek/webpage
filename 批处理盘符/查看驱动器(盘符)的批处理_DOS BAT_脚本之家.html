查看驱动器(盘符)的批处理

发布：mdxy-dxy 字体：[增加 减小] 类型：转载
用来查看驱动器信息的命令，方便我们了解当前磁盘情况。
复制代码 代码如下:

@echo off
setlocal enabledelayedexpansion
for /f "skip=1 tokens=1,2 delims=\" %%a in ('fsutil fsinfo drives^|find /v ""') do (
set aa=%%a
set aa=!aa:~-2!
for %%i in (!aa!) do fsutil fsinfo drivetype %%i
)
pause
