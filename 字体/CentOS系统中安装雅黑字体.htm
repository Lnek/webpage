在CentOS系统中安装雅黑字体

[日期：2011-09-28] 来源：Linux社区  作者：Linux [字体：大 中 小]

有很多人有自己经常使用的字体，在CentOS系统中我经常使用的就是雅黑，所以，我就以雅黑为例，来讲解在如何在CentOS系统中安装字体的。 CentOS系统下安装字体雅黑,Consolas文鼎字体，和文泉驿固然还可，但是习惯上，还是觉得不是太美观，Ubuntu的群体里偏向使用雅黑。我目前用的YaHei.Consolas 1.11 版本（雅黑－Consolas的混合体）

YaHei.Consolas 1.11 字体下载在：

在Linux公社的1号FTP服务器里，下载地址：

FTP地址：ftp://www.linuxidc.com

用户名：www.linuxidc.com

密码：www.muu.cc

在 2011年LinuxIDC.com\9月\在CentOS系统中安装雅黑字体

下载方法见这里 http://www.linuxidc.net/thread-1187-1-1.html

把雅黑的ttf文件拷到/usr/share/fonts/的任意目录下，假设雅黑ttf叫做Yahei.ttf：

sudo mkdir -p /usr/share/fonts/vista
sudo cp Yahei.ttf /usr/share/fonts/vista/

然后，改变权限：

sudo chmod 644 /usr/share/fonts/vista/*.ttf

安装：

cd /usr/share/fonts/vista/
sudo mkfontscale
sudo mkfontdir
sudo fc-cache -fv

最后，去CentOS系统－首选项－字体 里修改 Ubuntu字体。

在CentOS系统装微软雅黑字体 For Linux

2007-10-20 18:28（Windows XP 安装微软雅黑字体请看另一片文章：http://hi.baidu.com/red%5Fwoods/blog/item/a391b8367d2a7f390b55a930.html）

1. 安装字体

sudo mkdir /usr/share/fonts/msyh
sudo cp msyh.ttf msyhbd.ttf /usr/share/fonts/msyh/
sudo fc-cache -fv

2. 根据需要创建/修改字体配置文件 "/etc/fonts/local.conf" 或 "~/.fonts.conf"

字体配置文件可以对字体做详细设置. 也可以直接使用这个网上找的字体配置文件:
sudo cp local.conf ms_vista_english.conf ms_vista_cjk.conf /etc/fonts/

3. 在图形界面设置字体

最后在Gnome的字体管理工具中, 设置字体为微软雅黑, 以及字体大小, 字体渲染, DPI等, 以调整为最佳效果.

这样，我们就可以在CentOS系统中使用雅黑字体了，别的字体跟这个方法类似。快来选择你喜欢的字体吧！
