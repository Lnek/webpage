试看春残花渐落，便是红颜老死时!

豫东之花

2012-09-07 21:04

Windows Server 2012 主题

1. 按WIN+R 打开运行，输入services.msc回车，找到Themes 服务项 ，  右键点属性 把启动类型改为“自动”

 

新装的系统是没有像WIN8这样的主题的   

方法是    ：  打开服务器管理器 -> 添加角色和功能 -> 点“服务器选择”-> 再点“功能”-> 在右面找到“用户界面和基础构造”展开 -> 把“桌面体验”打钩    点击“安装”。同时会装两个必要组件“墨迹和手写服务”和“Media Foundation”。直接确定。安装完成后重启电脑。

 

 

 

不过  不再会出现那个玻璃透明效果了
