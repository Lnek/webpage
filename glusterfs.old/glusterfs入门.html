glusterfs入门

(2011-01-14 18:25:35)
转载▼
标签：

gluster

入门

安装

配置

挂载

it

 分类： 开源应用
一、简单介绍

1，内置支持nfs v3协议，可以运行在所有兼容POSIX的操作系统上，例如GUN/Linux
产品分为
gluster fs
gluster stroage

2，系统需求
推荐ext4文件系统（kernel 2.6.31以后），或者ext3早期版本。其他例如XFS和ZFS也可以，不过没有经过测试

3，安装文档
http://www.gluster.com/community/documentation/index.php/Gluster_3.1_Filesystem_Installation_and_Configuration_Guide
http://www.gluster.com/community/documentation/index.php/Gluster_3.1_Storage_Platform_Installation_Guide

二、安装

1，前提
yum -y install fuse fuse-libs wget

2，可选的步骤(普通用户非万M跳过，不要看）
安装inifiniband运行命令
ibv_srq_pingpong

wget http://www.openfabrics.org/downloads/OFED/ofed-1.5.2/OFED-1.5.2.tgz
tar xvfz OFED-1.5.2.tgz
cd OFED-1.5.2
./install.pl

报错：
libevent-devel is required to build rnfs-utils.
nfs-utils-lib-devel is required to build rnfs-utils.

然后安装
yum install libevent-devel

yum install tcl-devel
yum install libsysfs-devel


配置信息/etc/infiniband/info

验证IB Fabric
启动：/etc/init.d/openibd start
ibv_devinfo命令查看IB驱动信息

/etc/init.d/opensmd status

/etc/opensm/opensm.conf





3，端口111，38465+服务器数量的端口是开放状态

4，下载GlusterFS和FUSE RPM包

#git clone git://git.sv.gnu.org/gluster.git glusterfs
或
#wget http://download.gluster.com/pub/gluster/glusterfs/3.1/LATEST/CentOS/glusterfs-3.1.1-1.src.rpm
#wget http://download.gluster.com/pub/gluster/glusterfs/3.1/LATEST/CentOS/glusterfs-core-3.1.1-1.x86_64.rpm
#wget http://download.gluster.com/pub/gluster/glusterfs/3.1/LATEST/CentOS/glusterfs-fuse-3.1.1-1.x86_64.rpm
#wget http://download.gluster.com/pub/gluster/glusterfs/3.1/LATEST/CentOS/glusterfs-rdma-3.1.1-1.x86_64.rpm

以RPM包为例，安装
#rpm -vUh glusterfs-core-3.1.1-1.x86_64.rpm
#rpm -vUh glusterfs-fuse-3.1.1-1.x86_64.rpm

验证安装结果
#/usr/sbin/glusterfs -V

启动：#/etc/init.d/glusterd start

配置为自启动：#chkconfig glusterd on





三、初始配置

通常有以下3个

1，distributed：存放文件的分布式卷
其实只是扩大的磁盘空间，如果有一个磁盘坏了，对应的数据也丢失

2，distributed replicated：可以在2个或多个节点之间复制数据
high-availability and high-reliability
写性能高



3，distributed Striped：可以在2个或多个节点之间的Stripes文件
high concurrency environments accessing very large files

（一）配置分布式卷
1，创建glusterfs pool

#gluster peer probe 10.34.7.168
Probe successful
#gluster peer status           
Number of Peers: 1

Hostname: 10.34.7.168
Uuid: b3b3a61f-ddec-41f3-b0b5-09b6ecb20012
State: Peer in Cluster (Connected)

注意：不要加入本机，本机默认已经在pool里

2，创建分布式volume

#gluster volume create volume001 transport tcp 10.34.7.167:/linfei/gluster 10.34.7.168:/linfei/gluster
Creation of volume volume001 has been successful. Please start the volume to access data

启用卷
#gluster volume start volume001
Starting volume volume001 has been successful

查看信息
#gluster volume info all
Volume Name: volume001
Type: Distribute
Status: Started
Number of Bricks: 2
Transport-type: tcp
Bricks:
Brick1: 10.34.7.167:/linfei/gluster
Brick2: 10.34.7.168:/linfei/gluster

（二）配置分布式复制（镜像）卷

1，创建glusterfs pool
#gluster peer probe 10.34.7.168

2，创建volume
#gluster volume create volume002 replica 2 transport tcp 10.34.7.167:/linfei/gluster2 10.34.7.168:/linfei/gluster2

启用卷
#gluster volume start volume002


（三）配置分布式条带卷
1，创建glusterfs pool
#gluster peer probe 10.34.7.168

2，创建volume
跨2台机器的条带
#gluster volume create volume003 stripe 2 transport tcp 10.34.7.167:/linfei/gluster2 10.34.7.168:/linfei/gluster2

启用卷
#gluster volume start volume002


（四）其它可选项配置
1，添加CIFS Protocol
vi smb.conf

[test]
comment = Sample export via SAMBA
valid users = user3
printable = no
writable = yes
public = no
path = /mnt/gluster/user3_data
posix locking = yes

/mnt/gluster是挂载点

四、文件系统日常管理
http://www.gluster.com/community/documentation/index.php/Gluster_3.1_Filesystem_Administration_Guide

五、访问卷

三种方式
1，NFS

2，CIFS

3，gluster native client：推荐这个，这里也只介绍这个
可以确保high concurrency and high performance

（一）gluster native client
1，安装
前提


安装依赖包
#yum -y install fuse fuse-libs openib libibverbs wget openssh-server

设置iptables防火墙策略

下载GlusterFS core, FUSE, and RDMA RPM包到每个client上
#wget http://download.gluster.com/pub/gluster/glusterfs/3.1/LATEST/CentOS/glusterfs-core-3.1.1-1.x86_64.rpm
#wget http://download.gluster.com/pub/gluster/glusterfs/3.1/LATEST/CentOS/glusterfs-fuse-3.1.1-1.x86_64.rpm
#wget http://download.gluster.com/pub/gluster/glusterfs/3.1/LATEST/CentOS/glusterfs-rdma-3.1.1-1.x86_64.rpm

安装
rpm -vUh glusterfs-core-3.1.1-1.x86_64.rpm
rpm -vUh glusterfs-fuse-3.1.1-1.x86_64.rpm
rpm -vUh glusterfs-rdma-3.1.1-1.x86_64.rpm

最后
加载FUSE内核模块到Linux内核
#modprobe fuse
验证FUSE模块
#dmesg | grep -i fuse
fuse init (API version 7.10)


2，人工挂载卷

mount -t glusterfs 10.34.7.167:/volume001 /opt/glusterfs

其它端口服务器挂载
mount -t glusterfs 10.34.7.167:7777:/volume001 /opt/glusterfs


3，自动挂载
vi /etc/fstab
添加
10.34.7.167:/volume001 /opt/glusterfs glusterfs defaults,_netdev 0 0

4，验证挂载结果
#mount
glusterfs#10.34.7.167:/volume001 on /opt/glusterfs type fuse (rw,allow_other,default_permissions,max_read=131072)
#df
glusterfs#10.34.7.167:/volume001 60G  1.2G   55G   3% /opt/glusterfs

六、文件系统管理

1，添加资源池服务器
#gluster peer probe 10.34.7.165


2，卷里新增机器
#gluster volume add-brick volume001 10.34.7.166:/linfei/gluster 10.34.7.168:/linfei/gluster
#gluster volume remove-brick volume001 10.34.7.165:/linfei/glusterfs 10.34.7.168:/linfei/gluster

3，rebalance卷
当扩展或者收缩一个卷（add-brick/remove-brick）以后，需要rebalance数据
#gluster volume rebalance volume001 start

检查rebalance
#gluster volume rebalance volume001 status

停止rebalance
#gluster volume rebalance volume001 stop

4，修改volume配置
#gluster volume set volume001 option parameter

注：如果你复制模式和条带模式是2，那么你添加和删除volume里的资源一定要是2的倍数
