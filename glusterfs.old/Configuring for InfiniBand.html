Configuring for InfiniBand¶

You can optionally configure GlusterFS for OpenFabrics verbs RDMA-conformant transports, such as InfiniBand, using Red Hat Package Manager (RPM) distributions.

To configure GlusterFS for InfiniBand on RPM distribution-based systems

  1. Install required prerequisites on the server using the following command:

    $ sudo yum -y install openib libibverbs

  2. Install OFED version 1.5.2 from source using the directions in the download -

    $ wget http://www.openfabrics.org/downloads/OFED/ofed-1.5.2/

  3. Install GlusterFS.

    For more information, see Installing GlusterFS Using Red Hat Package Manager (RPM) Distributions.

  4. Using SSH, download the RDMA RPM file to each server in your cluster.

    The RDMA packages contain the OpenFabrics verbs RDMA module for Infiniband and 10GigE networks.

    For CentOS 5, download the file using the following command:

    $ wget ftp://ftp.gluster.com/pub/gluster/glusterfs/qa-releases/3.1/GlusterFS/CentOS/glusterfs-rdma-3.1.0alpha-1.x86_64.rpm

    For Fedora 11, download the file using the following command:

    $ wget ftp://ftp.gluster.com/pub/gluster/glusterfs/qa-releases/3.1/GlusterFS/Fedora11/glusterfs-rdma-3.1.0alpha-1.fc11.x86_64.rpm

  5. Install the RDMA module on the server using the following command:

    $ sudo rpm -Uvh <rdma_RPM_file>

    For example:

    $ sudo rpm -Uvh glusterfs-rdma-3.1.0alpha-1.x86_64.rpm
