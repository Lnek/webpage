Install Oracle JDK 8 on CentOS 6.X

The Guide Show in Easy-Steps How-to Install and Getting-Started with the Latest Oracle-Sun Official Java SE JDK 8 i586/x8664 for CentOS 6.1/6.2/6.3/6.4/6.5 i686/x86_64 Linux Desktop/Server.

To SetUp You will need to Use the update-alternatives Tool to Update your System Java to 1.8 JDK8.

The Powerful Java 8 New Features:

  • LAMBDA Expressions Programming
  • The Stream API
  • The Nashorn Javascript Engine

To the How-to Install Oracle JDK 8 on CentOS 6.X Bottom there is a Link for Quick Getting-Started with Java JDK 8 Development and also to Install the Java Web-Server Tomcat 7 for CentOS Linux.

To Make the Entire Process Easier to Visualize and Retain the Post Includes the Basic Screenshots of the Essentials Steps Involved.

The Content and Details of How-to Install Oracle JDK 8 on CentOS are Expressly Reduced to Give Focus Only to the Essentials Instructions and Commands and Make Tut Easier to Read & Digest ;)

  1. Download Oracle Java SE JDK8 for Linux:

    Oracle JDK8 i586/x86_64 tar.gz
    Get Oracle JDK8 .tar.gz.

  2. Double-Click/Righ-Click on Archive and Extract into /tmp

  3. Open a Terminal Window

    Or from Shell:

    tar xvfz jdk-8*.tar.gz
  4. Relocate JDK 8 
    sudo su
    if [ ! -d '/usr/lib/jvm' ]; then mkdir /usr/lib/jvm; fi
    mv /tmp/jdk1.8* /usr/lib/jvm/

  5. Installing Oracle Java JDK 8

    update-alternatives --install /usr/bin/java java /usr/lib/jvm/jdk1.8*/bin/java 1065

    update-alternatives --install /usr/bin/javac javac /usr/lib/jvm/jdk1.8*/bin/javac 1065

    update-alternatives --install /usr/bin/jar jar /usr/lib/jvm/jdk1.8*/bin/jar 1065

    update-alternatives --install /usr/bin/javaws javaws /usr/lib/jvm/jdk1.8*/bin/javaws 1065

    Checking or Setting the System Java Version in Use with

    update-alternatives --config java

    To Switch of Java Version Just Enter the Selection Number on First Column. :)

  6. Test New Java JDK 8 Installation

    java -version

  7. How-to Install Eclipse with Java 8 Support on CentOS

    Eclipse with JDK8 Support
    Install Eclipse for Java 8.

    8. Now Thanks to Oracle-Sun Corporation You are Able to Successfully Start Up with Java 8 Development.
