Solved the problem this way (running Eclipse 3.7 on Ubuntu 12.04) : On the installation dialog, I unchecked the box "Show only the latest versions of available software" and selected the PyDev 2.8.2 version, which properly installed. I previously tried 3.0, 3.1 and 3.2, all failed.
